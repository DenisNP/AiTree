# ИИ-гирлянда на адресных светодиодах

[Демо.](https://t.me/clockstackwheels/1197)

Описываете Алисе сцену, а она раскрашивает ёлку в палитру этой сцены. Нужны знания того, как создать приватный навык для Алисы + отдельный сервер с SSL.

Прошивается на ESP32. Все важные настройки лежат в `AiTreeFirwmare/include/globals.h`.

ИИ-часть работает под управлением DeepSeek, нужно в переменные окружения вставить `DEEPSEEK_API_KEY`.
